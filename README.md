Customer Segmentation Analysis Using Data Analytics

This project analyzes customer behavior and segments customers based on income, spending score, and purchasing patterns using Python and Power BI.

Tools Used

- Python (Jupyter Notebook)
- Power BI
- Excel

Features

- Customer clustering using K-Means algorithm
- KPI cards for customer insights
- Gender distribution analysis
- Average spending analysis by customer cluster
- Interactive Power BI dashboard
- Customer segmentation scatter plot

KPIs Included

- Total Customers
- Average Income
- Average Spending Score

Visualizations

- Pie Chart
- Bar Chart
- Scatter Plot
- Cluster Analysis Dashboard

Business Insights

- Identified high-value customers
- Analyzed customer spending behavior
- Improved understanding of customer segments

Files Included

- Power BI Dashboard (.pbix)
- Python Notebook (.ipynb)
- Dataset (.csv)
- Dashboard Screenshots
